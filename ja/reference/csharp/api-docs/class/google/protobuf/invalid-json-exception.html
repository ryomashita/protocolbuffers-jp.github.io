Google.Protobuf.InvalidJsonException

Thrown when an attempt is made to parse invalid JSON, e.g.

Summary

using a non-string property key, or including a redundant comma. Parsing a protocol buffer message represented in JSON using JsonParser can throw both this exception and InvalidProtocolBufferException depending on the situation. This exception is only thrown for "pure JSON" errors, whereas InvalidProtocolBufferException is thrown when the JSON may be valid in and of itself, but cannot be parsed as a protocol buffer message.
